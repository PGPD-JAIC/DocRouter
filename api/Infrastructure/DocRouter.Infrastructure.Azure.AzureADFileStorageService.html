    Task<Submission>

    A object with the relevant fields from the MSGraph action populated.
    Remarks

    The following fields of the submission are required to create a directory:

    • Submission.TitleUsed as the title of the folder created in the document library.
    • Submission.Transactions[0].RoutedToUsed to set access permissions on the DriveItem.
    • Submission.Transactions[0].RoutedFromUsed to set access permissions on the DriveItem.
    • Submission.DriveIdRequired for the API call.
    • Submission.ListIdRequired for the API call.
